# VelvetTable
## Smart Restaurant Reservation & Pre-Order Dining Platform

VelvetTable is a modern web platform designed to make dining **instant, easy, and hassle-free**.  
Users can **reserve tables**, **pre-order food**, and **check in using QR codes**, ensuring that both the table and the food are ready exactly when they arrive.

---

## Features

### Restaurant Experience
- Browse restaurants with **real-time available time slots**
- View menus and restaurant details
- Reserve a table instantly

### Pre-Order System
- Pre-order menu items before visiting
- Optional **online pre-payment**
- Significantly reduces waiting time at the venue

### Smart Check-In
- Auto-generated **QR code** for each booking
- Quick and secure validation at the restaurant

### User Profile & History
- Create and manage a personal profile
- View past and upcoming reservations
- Track order history and preferences

### UI / UX
- Clean, minimal, and responsive design
- Focus on fast navigation and smooth user flow

### AI-Assisted Development
- Leveraged AI tools for faster prototyping, debugging, and feature refinement
